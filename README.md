# CodeAlpha_Login-and-Registration-System
Description
A simple C++ console application that allows users to register and log in using a username and password. User credentials are stored in a file and verified during login.

Features
User Registration
User Login
Input Validation
Duplicate Username Checking
File-Based Storage
Success and Error Messages

Technologies Used
C++
File Handling (ifstream, ofstream)
Functions and Conditional Statements

How It Works
Register with a username and password.
Credentials are saved in users.txt.

Login using the registered credentials.
The system verifies the information and displays the result.

Conclusion
This project demonstrates basic authentication, file handling, and input validation concepts in C++.
